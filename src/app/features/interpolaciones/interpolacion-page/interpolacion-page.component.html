Qué es la interpolación en Angular

La interpolación es una forma de mostrar datos del componente en la plantilla HTML usando la sintxis {{expresion}}

  • Permite insertar valores dinámicos desde el componente al HTML.
  • Se usa para mostrar texto, variables o resultados de expresiones simples.
  • No se debe usar para llamadas a funciones que modifiquen datos (efectis secundarios)

Ejemplo simple de interpolación:

{{ nombre}}
